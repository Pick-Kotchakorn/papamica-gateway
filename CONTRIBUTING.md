# Contributing to Papamica Gateway

##  Overview
This project uses a LINE Bot with Google Apps Script (GAS) backend and Cloudflare Workers.

##  Prerequisites
- Node.js 18+
- Git
- Google Account
- Cloudflare Account

##  Getting Started

### Step 1: Clone
`git clone https://github.com/Pick-Kotchakorn/papamica-gateway.git`

### Step 2: Install
`npm install -g @google/clasp`

##  Git Workflow

- main  Production
- feature/*  New features
- bugfix/*  Bug fixes

##  Commit Convention

feat(scope): description
fix(scope): description
docs: description

Happy coding!
